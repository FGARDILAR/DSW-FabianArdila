Proposito del Sitio



El Objetivo principal del sitio es dar a conocer que son las Hojas de Estilo en Cascada CSS y los Frameworks CSS, su definición y dar una breve explicación de sus importancia y usos en el desarrollo de paginas WEB.





Introducción



Las hojas de estilo en cascada constituyen una gran herramienta para la construcción de páginas web. Su función es la de definir el formato, presentación y diseños de los diferentes elementos que integran un documento.

Asi mismo los framework de CSS es una biblioteca de estilos genéricos que puede ser usada para implementar diseños web. Aportan una serie de utilidades que pueden ser aprovechadas frecuentemente en los distintos diseños web.
